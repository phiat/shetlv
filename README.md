# shetlv
(curl) ->  (cat | pup | paste) -> (psql)   🤠

# example usage: 
`shetlv.sh cbb_conf etlv`

mix and match 'etlv' as needed  
* (e)xtract 
* (t)ransform
* (l)oad
* (v)iew

# requirements:  
[curl](https://curl.haxx.se/), [pup](https://github.com/ericchiang/pup), [postgresql](https://www.postgresql.org/) (with user as superuser)
